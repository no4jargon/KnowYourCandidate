
  This is a code bundle for a responsive web app tool that assists business higher the best candidate for a job

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run server` in one terminal to start the authentication API (defaults to port `3000`).

  Run `npm run dev` in another terminal to start the Vite development server on port `5173`. The Vite dev server proxies requests from `/api` to the Express API, so no additional configuration is required for local development.
